# 📚 Student Study Planner

A modern web-based Study Planner built using Flask and MySQL that helps students organize their study tasks, track progress, and improve productivity.

---

## 🚀 Features

- User Registration & Login
- Secure Password Hashing
- Session-based Authentication
- Dashboard with Statistics
- Add New Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Search Tasks
- Profile Page
- Progress Tracking
- Flash Messages
- Responsive Design

---

## 🛠 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Flask

### Database
- MySQL

### Libraries
- Flask
- Flask Sessions
- Werkzeug Security
- MySQL Connector

## ⚙ Installation

```bash
git clone <repository-url>
```

```bash
cd Student-Study-Planner
```

```bash
pip install -r requirements.txt
```

Create a MySQL database and update your database credentials in `app.py`.

Run the project:

```bash
python app.py
```
